# Mahindra CIE QR Automation

Python-based QR Automation Software developed for Mahindra CIE.

## Features

- Excel file upload
- Automatic QR code generation
- Single QR for multiple entries
- PDF export with company format
- GUI-based desktop software
- Dynamic timestamp-based outputs

## Technologies Used

- Python
- Tkinter
- Pandas
- qrcode
- ReportLab

## How to Run

Install dependencies:

pip install -r requirements.txt

Run GUI:

python gui.py

## Output

- QR image generated automatically
- PDF labels generated inside output folder
